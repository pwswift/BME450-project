## Project decription

- We plan to utilize a dataset of coronal CT scans of kidneys. It is a collection of medical images for spotting kidney stones. The dataset features various kidney stone sizes, shapes, and positions within the urinary system. This dataset was sourced from Kaggle (https://www.kaggle.com/datasets/safurahajiheidari/kidney-stone-images/data). 

- The goal of the project is to first identify the presence of kidney stones within a renal/urinary system using a neural network. Assuming the model is able to accurately determine when a stone is present based on the CT scan, this network could further be refined to determine the placement (kidney, ureter, etc.) within the renal system.

- Other refinements could include the use of more rigorous architectures, including convolutional neural network frameworks specifically developed for medical image segmentation and classification.
